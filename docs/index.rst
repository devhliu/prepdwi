Prepdwi Documentation
=====================

Prepdwi is a pipeline developped by the Khan Lab to pre-process Diffusion MRI data which is already in the BIDS format to perform tracktography. pre-processing includes denoising, unring, top-up, eddy, bedpost, etc.
